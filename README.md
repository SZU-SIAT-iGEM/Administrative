### 操作指南

#### 访问权限说明

**本文件夹部分内容涉及隐私信息，仅限特定人员访问**：

| 文件夹 | 访问权限 | 说明 |
|--------|----------|------|
| `团队管理/` | 全体成员 | 通用信息 |
| `财务记录/` | 队长、财务负责人 | 涉及金额明细 |
| `赞助与合作/` | 队长、外联负责人 | 涉及合作条款 |
| `注册与提交/` | 队长、副队长 | 涉及账号信息 |

#### 财务记录规范

1. 所有支出需有凭证
2. 凭证命名：`YYYY-MM-DD_项目_金额.jpg`
3. 每月生成财务报告
4. 预算变更需记录原因
